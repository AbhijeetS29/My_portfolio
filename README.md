## How to Run Locally  
1. Clone this repository:  
   ```bash
   git clone https://github.com/AbhijeetS29/My_portfolio.git
   ```  
2. Navigate to the project directory:  
   ```bash
   cd My_portfolio
   ```  
3. Install dependencies:  
   ```bash
   flutter pub get  
   ```  
4. Run the project:  
   ```bash
   flutter run -d web  
   ```
